Privacy Policy

StickyNative Privacy Policy

StickyNative は、メモの作成と管理をローカルで完結することを重視して設計されています。 本ポリシーは、アプリが取り扱う情報とその利用方法について説明するものです。

Support Page

1. 取得・保存する情報

StickyNative は、以下の情報をこの Mac のローカル環境に保存します。

  • ユーザーが作成したメモ本文
  • メモのタイトル、固定状態、ゴミ箱状態、セッション情報
  • メモウィンドウの位置、サイズ、開閉状態
  • フォントサイズや新規メモサイズなどのアプリ設定

2. 保存場所

メモデータは SQLite データベースとして Application Support/StickyNative 配下に保存されます。 一部の設定値は UserDefaults に保存されます。

3. 収集しない情報

StickyNative は、通常利用において以下のような情報を収集しません。

  • 氏名、メールアドレス、電話番号などの個人情報
  • 位置情報、連絡先、カレンダー、写真、マイク、カメラのデータ
  • 広告識別子、トラッキング情報、解析イベント
  • メモ内容を外部サーバーへ送信する処理

4. 第三者提供

StickyNative は、アプリ内で保存されたメモや設定を第三者へ販売、共有、提供しません。

5. インターネット接続について

StickyNative の通常利用にインターネット接続は不要です。 アプリの基本機能はローカルで完結します。

6. サポート連絡時の情報

ユーザーが GitHub Issues などを通じて問い合わせを行った場合、その際にユーザー自身が送信した情報は サポート対応のために利用されることがあります。

7. 本ポリシーの変更

今後の機能追加や配布形態の変更に応じて、本ポリシーを更新する場合があります。 重要な変更がある場合は、公開ページ上で内容を更新します。

8. お問い合わせ

サポートや不具合報告については、GitHub Issues をご利用ください。
