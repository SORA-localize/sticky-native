StickyNative

macOS のメニューバーに置ける、軽量な付箋アプリ。

StickyNative は、必要なときにすぐ開いて書けることを重視した macOS 用メモアプリです。メニューバー常駐、キーボードショートカット、 ローカル保存を中心に、作業の流れを止めない短いメモ体験を目指しています。

GitHub Repository Privacy Policy

主な機能

  • メニューバーからすぐに新規メモを作成
  • グローバルショートカットで素早く呼び出し
  • メモ内容の自動保存と再起動後の復元
  • 常時前面表示の切り替え
  • 一覧画面での管理、ゴミ箱、復元、完全削除
  • セッションごとの整理
  • フォントサイズと新規メモサイズの設定

使い方

  1. アプリを起動すると、メニューバーに StickyNative が表示されます。
  2. メニューバーのメニュー、またはグローバルショートカットから新しいメモを作成できます。
  3. 入力した内容は自動保存されます。メモウィンドウ上では保存や保存して閉じるショートカットも使えます。
  4. All Memos からすべてのメモを管理し、セッション分け、ゴミ箱への移動、復元、完全削除ができます。

よくある質問

データはどこに保存されますか？

メモデータはこの Mac のローカルストレージに保存されます。 現在の実装では SQLite データベースを Application Support/StickyNative 配下に作成し、 設定値は UserDefaults に保存します。

インターネット接続は必要ですか？

通常利用にインターネット接続は不要です。StickyNative はメモの作成、 保存、管理をローカルで完結する設計です。

ショートカットが動かない場合は？

macOS の環境や権限状態によっては挙動が変わることがあります。 まずアプリを再起動し、最新版を利用しているか確認してください。
